3 data block styles

First grid layer is built with .data-block cells. Second one works inside of data blocks. Use standard Bootstrap grid classes with data-block cell to build your own layout. Each .data-block comes with optional components, you can remove <footer> and <header> if needed or turn whole block into more complex widget.

You can choose from 3 different styles for each data block. Default style drops fancy papier-like shadows. For simpler look use class .decent or alternatively clean all styling with class .raw.

Check these 3 blocks for example configuration.

	<article class="spanX data-block">
		<div class="data-container">
			<header>
				<h2>Headline</h2>
			</header>
			<section>
				Content
			</section>
			<footer>
				Footer note
			</footer>
		</div>
	</article>
